사무 자동화 웹 서비스

반복 업무를 메뉴별로 빠르게

로그인 없이 바로 진입할 수 있고, 엑셀 변환부터 인사/근태/급여 분석, 게시판까지 한 화면에서 접근할 수 있도록 구성했습니다.

현재 운영 중인 핵심 자동화 기능 엑셀 2종 + 인사 1종 + 근태 1종

실제 사용 가능한 기능만 노출해 서비스 구조를 단순하게 유지합니다.

빠른 실행

메인 대시보드

업무 카테고리를 선택하세요

운영 원칙

실제 동작하는 도구만 공개하고, 업로드 파일은 브라우저 내 처리 또는 즉시 결과 생성 중심으로 동작합니다.

콘텐츠 기준

도구 설명, 이용 방법, 개인정보처리방침, 문의 채널을 함께 제공해 서비스 목적과 운영 주체를 명확히 표시합니다.

광고 노출 기준

광고는 본문과 기능 사용을 방해하지 않는 영역에만 배치하고, 메뉴나 다운로드 버튼처럼 오인될 수 있는 위치에는 두지 않습니다.

엑셀 파일 변환

파일 병합

여러 엑셀 파일을 업로드하고, 컬럼 행 위치와 병합 방식을 지정해 하나의 결과 파일로 내려받을 수 있습니다.

병합 방식
파일을 선택한 뒤 병합 파일 생성을 실행하세요.

엑셀 파일 변환

파일 분할

여러 엑셀 파일을 업로드한 뒤 행 기준, 컬럼 기준, 시트 기준으로 분할해 ZIP 파일로 다운로드할 수 있습니다.

분할 방식
파일을 선택한 뒤 분할 파일 생성을 실행하세요.

서비스 안내

서비스 소개

이 서비스는 반복적인 엑셀 사무 업무를 줄이기 위해 제작된 웹 기반 자동화 도구입니다.

현재 제공 기능은 파일 병합, 파일 분할, 인원 분석, 연차관리대장 제작입니다. 각 기능은 업로드한 파일을 기준으로 결과 파일을 즉시 생성하는 형태로 동작합니다.

미구현 메뉴는 승인 전 노출하지 않고 있으며, 운영 중인 도구의 범위와 처리 방식을 화면에서 바로 확인할 수 있도록 구성했습니다.

인사 정보 관련

인원 분석

인사 엑셀 파일을 업로드하고 컬럼 행과 기준 연월을 입력하면, 해당 월의 입사자/퇴사자와 말일 기준 재직 인원을 정리한 엑셀 파일을 생성합니다.

파일과 기준 연월을 입력한 뒤 분석 파일 생성을 실행하세요.

근태 관련

연차관리대장 제작

인사 엑셀 파일을 업로드하고 생성 연도와 기준 방식을 선택하면, 1월부터 12월까지 연결된 연차관리대장 엑셀 파일을 생성합니다.

전년도 중도 입사자의 미사용 월차 이월분은 자동 계산하지 않습니다. 전년도 월차는 연말까지 소진 완료한 기준으로 작성되며, 이월이 필요한 값은 생성 파일의 `전기잔여` 컬럼에 직접 입력해야 합니다.
연차 생성 기준
파일과 생성 연도, 기준 방식을 입력한 뒤 연차관리대장 생성을 실행하세요.

서비스 안내

이용 안내

1. 파일 병합

여러 엑셀 파일을 하나의 시트 또는 여러 시트로 합칩니다. 컬럼이 다르면 새로운 컬럼을 오른쪽에 추가합니다.

2. 파일 분할

행 수 기준, 컬럼 값 기준, 시트 기준으로 엑셀 파일을 분할하고 ZIP으로 제공합니다.

3. 인원 분석

기준 연월을 입력하면 입사자, 퇴사자, 말일 기준 재직 인원을 분석한 엑셀 파일을 반환합니다.

4. 연차관리대장 제작

선택 연도 기준으로 1월부터 12월까지 연결된 연차관리대장 시트를 생성합니다. 사용일 입력 시 당월사용과 잔여연차가 자동 반영됩니다.
