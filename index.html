컬럼 매핑과 미리보기 차트 패널을 확장할 수 있습니다.

인사 정보 관련

인원 분석

인사 엑셀 파일을 업로드하고 컬럼 행과 기준 연월을 입력하면, 해당 월의 입사자/퇴사자와 말일 기준 재직 인원을 정리한 엑셀 파일을 생성합니다.

파일과 기준 연월을 입력한 뒤 분석 파일 생성을 실행하세요.

근태 관련

연차관리대장 제작

인사 엑셀 파일을 업로드하고 생성 연도와 기준 방식을 선택하면, 1월부터 12월까지 연결된 연차관리대장 엑셀 파일을 생성합니다.

전년도 중도 입사자의 미사용 월차 이월분은 자동 계산하지 않습니다. 전년도 월차는 연말까지 소진 완료한 기준으로 작성되며, 이월이 필요한 값은 생성 파일의 `전기잔여` 컬럼에 직접 입력해야 합니다.
연차 생성 기준
파일과 생성 연도, 기준 방식을 입력한 뒤 연차관리대장 생성을 실행하세요.

급여 관련

급여 비교분석

월별 또는 부서별 급여 데이터를 비교하고 차이를 분석하는 영역입니다.

비교 대상 기간 선택과 증감률 지표 카드 구성이 적합합니다.

급여 관련

사업계획

인건비 기준 사업계획 수립을 위한 시뮬레이션과 예산 산출 영역입니다.

조직별 인원 계획과 예산 추정 테이블을 연결할 수 있습니다.

급여 관련

퇴직금 계산

근속 기간과 평균 임금을 기준으로 퇴직금을 계산하는 기능 영역입니다.

계산 기준 입력 폼과 결과 요약 카드가 들어갈 자리입니다.
