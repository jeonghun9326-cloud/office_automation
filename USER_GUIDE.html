사무 자동화 서비스 사용방법

본 문서는 현재 서비스에 구현된 기능 기준으로 작성되었습니다. 각 기능은 엑셀 파일을 업로드하고 결과 파일을 다운로드하는 방식으로 동작합니다.

1. 파일 병합

  1. `엑셀 파일 변환 > 파일 병합` 메뉴로 이동합니다.
  2. 병합할 엑셀 파일을 여러 개 선택합니다.
  3. 컬럼이 있는 행 번호를 입력합니다. 첫 행이 컬럼이면 `1`을 입력합니다.
  4. 병합 방식을 선택합니다.
  • 하나의 시트에 병합: 서로 다른 컬럼은 오른쪽 끝에 추가되고, 데이터는 순차적으로 이어집니다.
  • 시트 구분 병합: 업로드 파일별로 각각 다른 시트로 생성됩니다.
  1. `병합 파일 생성` 버튼을 누릅니다.
  2. 생성된 엑셀 파일을 다운로드합니다.

2. 파일 분할

  1. `엑셀 파일 변환 > 파일 분할` 메뉴로 이동합니다.
  2. 분할할 엑셀 파일을 여러 개 선택합니다.
  3. 컬럼 행 번호를 입력합니다.
  4. 분할 방식을 선택합니다.
  • 행 기준 분할: 지정한 행 수 단위로 파일을 나눕니다.
  • 컬럼 기준 분할: 입력한 컬럼명 값이 같은 행끼리 파일을 나눕니다.
  • 시트 기준 분할: 각 시트를 별도 파일로 분리합니다.
  1. 필요한 옵션을 입력합니다.
  2. `분할 파일 생성` 버튼을 누릅니다.
  3. 결과는 ZIP 파일로 다운로드됩니다.

3. 인원 분석

기준 월의 입사자, 퇴사자, 말일 기준 재직 인원을 정리한 엑셀 파일을 생성합니다.

  1. `인사 정보 관련 > 인원 분석` 메뉴로 이동합니다.
  2. 인사 엑셀 파일을 업로드합니다.
  3. 컬럼 행 번호를 입력합니다.
  4. 기준 연월을 선택합니다.
  5. `분석 파일 생성` 버튼을 누릅니다.

필수 컬럼

  • 사번
  • 성명 또는 이름
  • 입사일 또는 입사
  • 퇴사일 또는 퇴사

주의사항

  • 같은 사번이 여러 번 존재하면 1명으로만 계산합니다.
  • 날짜 형식은 `yyyy-mm-dd`, `yyyy.mm.dd`, `yyyy년mm월dd일`, 숫자형 Excel 날짜 등을 최대한 인식합니다.

결과 파일 구성

  • 요약
  • 입사자
  • 퇴사자
  • 재직자

4. 연차관리대장 제작

선택 연도의 1월부터 12월까지 연차관리대장을 자동으로 생성합니다.

  1. `근태 관련 > 연차관리대장 제작` 메뉴로 이동합니다.
  2. 인사 엑셀 파일을 업로드합니다.
  3. 생성 연도를 입력합니다.
  4. 컬럼 행 번호를 입력합니다.
  5. `회계년도 기준` 또는 `입사일 기준` 중 하나를 선택합니다.
  6. `연차관리대장 생성` 버튼을 누릅니다.

필수 컬럼

  • 사번
  • 이름 또는 성명
  • 입사일 또는 입사
  • 퇴사일 또는 퇴사

결과 파일 구성

  • 1월부터 12월까지 총 12개 시트 생성
  • 각 시트에 `전기잔여`, `전월잔여`, `연차발생`, `월차발생`, `당월사용`, `잔여연차` 포함
  • 2월 이후 시트의 `전월잔여`는 전월 `잔여연차`를 참조합니다.
전년도 중도 입사자의 미사용 월차 이월분은 자동 계산하지 않습니다. 필요한 경우 생성 파일의 `전기잔여` 컬럼에 직접 입력해야 합니다.

5. 급여 비교분석

여러 급여 파일을 기준으로 기간별 증감액 비교 결과를 시트별로 생성합니다.

  1. `급여 관련 > 급여 비교분석` 메뉴로 이동합니다.
  2. 급여 엑셀 파일을 여러 개 업로드합니다.
  3. 컬럼 행 번호를 입력합니다.
  4. 기간 비교 컬럼명을 입력합니다. 예: `지급월`, `지급일`
  5. 비교 기간을 선택합니다. 예: 월간, 분기, 반기, 연간
  6. 비교 형태를 선택합니다. 예: 전체, 부서별, 직급별, 직군별, 개인별
  7. 비교할 급여 항목을 선택합니다.
  8. `비교 파일 생성` 버튼을 누릅니다.

필수 컬럼

  • 사번

비교할 급여 항목 선택 규칙

  • `전체선택` 버튼으로 한 번에 선택할 수 있습니다.
  • `전체선택` 버튼을 다시 누르면 전체해제됩니다.
  • 사번, 이름/성명, 성별, 주민번호, 생년월일, 부서, 직급, 직위, 직책, 직군, 기간 컬럼은 자동으로 제외됩니다.

기간 형식 인식

  • `yyyy-mm-dd`
  • `yyyy.mm.dd`
  • `yyyy/mm/dd`
  • `yyyy년mm월dd일`
  • 월 단위 문자열 또는 숫자형 Excel 날짜

결과 파일 구성

  • 비교 구간별로 시트 생성
  • 각 시트에는 이전 기간 금액, 현재 기간 금액, 증감액이 표시됩니다.

6. 공통 주의사항

  • 컬럼명은 공백이나 표현 차이가 있을 수 있으므로 입력 전에 업로드 파일 헤더를 확인하는 것이 좋습니다.
  • 분석 결과는 업로드 파일 구조에 따라 달라질 수 있으므로 샘플 파일로 먼저 시험하는 것이 안전합니다.
  • 기능별로 일부 자동 계산이 어려운 항목은 결과 파일 안에서 직접 보정해야 할 수 있습니다.
