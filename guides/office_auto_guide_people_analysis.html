인원 분석 사용설명서

기준 기간의 입사자, 퇴사자, 재직 인원을 분석하고 요약 엑셀을 생성하는 기능입니다.

필수 컬럼

  • 사번
  • 성명 또는 이름
  • 입사일 또는 입사
  • 퇴사일 또는 퇴사

사용 순서

  1. 인사 파일을 업로드합니다.
  2. 컬럼 행 번호를 입력합니다.
  3. 분석 구분과 기준 연도, 기간을 선택합니다.
  4. 기간 컬럼명과 급여 컬럼명을 입력합니다.
  5. 분석 파일 생성을 누르면 결과 엑셀이 다운로드됩니다.

입력 안내

  • 기간 컬럼명과 급여 컬럼명은 정해진 이름이 없습니다. 실제 파일 컬럼명을 그대로 입력하면 됩니다.
  • 예시파일에 없는 추가 컬럼이 있어도 업로드에 문제 없습니다.
  • 같은 사번이 여러 번 있어도 인원 계산은 사번 기준으로 정리됩니다.

결과 파일 구성

  • 요약 시트
  • 입사자 시트
  • 퇴사자 시트
  • 재직자 시트
기간 컬럼에는 지급월, 급여월, 대상월처럼 실제 기간 판별이 가능한 컬럼을 넣는 것이 좋습니다.
