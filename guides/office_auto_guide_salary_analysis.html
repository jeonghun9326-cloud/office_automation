급여 비교분석 사용설명서

기간별 급여 항목의 증감과 그룹별 비교 결과를 엑셀로 정리하는 기능입니다.

필수 컬럼

  • 사번

사용 순서

  1. 급여 파일을 하나 이상 업로드합니다.
  2. 컬럼 행 번호를 입력합니다.
  3. 기간 비교 컬럼명을 입력합니다.
  4. 비교 기간, 비교 형태, 비교할 급여 항목을 선택합니다.
  5. 비교 파일 생성을 누르면 결과 파일이 다운로드됩니다.

입력 안내

  • 기간 비교 컬럼명은 정해진 이름이 없으며 실제 파일 컬럼명을 입력하면 됩니다.
  • 예시파일에 없는 추가 컬럼이 있어도 업로드할 수 있습니다.
  • 개인별 비교에서는 사번 컬럼이 반드시 필요합니다.

결과 파일 구성

  • 기간 구분별 비교 시트
  • 이전 기간, 현재 기간, 증감액 열
처음에는 비교 항목을 너무 많이 고르지 말고 대표 급여 항목부터 시험하는 것이 좋습니다.
