파일 분할 사용설명서

엑셀 파일을 행 기준, 컬럼 기준, 시트 기준으로 나눠 ZIP으로 받는 기능입니다.

사용 순서

  1. 분할할 파일을 업로드합니다.
  2. 컬럼 행 번호를 입력합니다.
  3. 분할 방식을 선택합니다.
  4. 필요한 추가 옵션을 입력합니다.
  5. 분할 파일 생성을 누르면 ZIP 파일이 다운로드됩니다.

분할 방식

  • 행 기준: 지정한 행 수 단위로 나눕니다.
  • 컬럼 기준: 특정 컬럼의 값별로 파일을 나눕니다.
  • 시트 기준: 각 시트를 개별 파일로 나눕니다.

주의사항

  • 컬럼 기준 분할에서 입력하는 컬럼명은 실제 헤더와 같아야 합니다.
  • 행 기준 분할에서는 데이터 행 개수만 계산됩니다.
  • 결과는 ZIP 압축 파일 하나로 제공됩니다.
