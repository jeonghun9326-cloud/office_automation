파일 병합 기능 구현 내용
- 다수의 엑셀 파일 업로드 가능
- 컬럼 행 번호 입력 가능
  - 예: 첫 번째 행이 컬럼이면 1
- 병합 방식 2가지 구현

1. 하나의 시트에 병합
- 업로드 파일들의 첫 번째 시트를 기준으로 처리
- 컬럼명이 서로 다를 수 있으므로 새 컬럼은 오른쪽 끝에 추가
- 없는 값은 빈 셀로 유지
- 컬럼 행 아래 데이터 행을 순차적으로 이어 붙임
- 결과 파일명: merged_single_sheet.xlsx

2. 시트 구분 병합
- 업로드 파일별로 하나의 시트 생성
- 시트명은 파일명을 기반으로 생성
- 결과 파일명: merged_by_sheets.xlsx

인원 분석 기능 구현 내용
- 메뉴명 변경
  - 기존: 입/퇴사 분석
  - 현재: 인원 분석
- 기존 재직 인원 분석 메뉴는 제거
- 인사 엑셀 파일 1개 업로드
- 컬럼 행 번호 입력
- 기준 연월 입력
- 필수 컬럼 검증
  - 사번 필수
  - 성명 또는 이름 중 1개 이상
  - 입사일 또는 입사
  - 퇴사일 또는 퇴사
- 중복 처리
  - 사번이 같은 행은 중복 카운팅하지 않음
  - 한 사람이 여러 번 기록되어 있어도 사번 기준으로 1회만 반영
- 결과 파일 생성
  - 요약 시트
  - 입사자 시트
  - 퇴사자 시트
  - 재직자 시트
- 계산 기준
  - 기준 월 내 입사일이면 당월 입사자
  - 기준 월 내 퇴사일이면 당월 퇴사자
  - 기준 월 말일 기준 입사 완료 + 퇴사 미발생(또는 말일 이후 퇴사)이면 재직자

연차관리대장 제작 기능 구현 내용
- 인사 엑셀 파일 1개 업로드
- 생성 연도 입력
- 컬럼 행 번호 입력
- 회계년도 기준 / 입사일 기준 선택
- 필수 컬럼 검증
  - 사번
  - 이름 또는 성명
  - 입사일 또는 입사
  - 퇴사일 또는 퇴사
- 날짜 형식은 다양한 표시형식을 인식하도록 보정
  - yyyy-mm-dd
  - yyyy.mm.dd
  - yyyy/mm/dd
  - yyyy년mm월dd일
  - 숫자형 Excel 날짜
- 중복 처리
  - 사번이 같은 행은 1명으로 처리
- 생성 결과
  - 선택 연도 1월~12월까지 12개 시트 생성
  - 1월~12월 모든 시트는 사번, 이름, 입사일, 퇴사일, 전기잔여, 전월잔여, 연차발생, 월차발생, 당월사용, 잔여연차, 날짜 컬럼 포함
  - 전월 잔여연차가 다음 월 시트에 수식으로 연결
  - 전년도 중도 입사자의 월차 이월분은 자동 계산하지 않음
  - 필요 시 전기잔여 컬럼에 사용자가 직접 입력
  - 1월 시트의 전월잔여는 0으로 고정

급여 비교분석 기능 구현 내용
- 다수의 급여 엑셀 파일 업로드
- 컬럼 행 번호 입력
- 비교할 급여 항목 다중 선택
- 비교 기간 선택
  - 월간비교
  - 분기비교
  - 반기비교
  - 연간비교
- 비교 형태 선택
  - 전체
  - 부서별
  - 직급별
  - 직군별
  - 개인별(사번 기준)
- 기간 비교 컬럼명 입력
- 필수 조건
  - 사번 컬럼 필수
- 부서/직급/직군 컬럼 탐지
  - 부서, 부서명
  - 직급, 직급명
  - 직군, 직군명
  - 앞뒤에 다른 문구가 있어도 부분 일치로 탐지
- 비교 항목 선택 UI
  - 전체선택 버튼 제공
  - 전체선택 버튼을 다시 누르면 전체해제
  - 사번/이름/성명/성별/주민번호/주민등록번호/생년월일/부서/직급/직위/직책/직군/기간 컬럼만 숨김
  - 그 외 컬럼은 모두 비교 항목으로 선택 가능
- 기간 컬럼 날짜 인식 보정
  - yyyy-mm-dd
  - yyyy.mm.dd
  - yyyy/mm/dd
  - yyyy년mm월dd일
  - yyyy-mm / yyyy.mm / yyyy년m월
- 결과
  - 비교 기간별로 시트 생성
  - 각 시트는 이전 기간, 현재 기간, 증감액을 급여 항목별로 표시
- 현재 구현 기준
  - 회계년도 기준은 1월에 연차를 부여
  - 전년도 중도 입사자는 15/12 * 전년도 재직 개월수로 계산
  - 1년 미만자는 최대 11회의 월차를 월별로 발생
  - 입사일 기준은 해당 연도의 입사일 도래 월에 연차를 발생
  - 사용일수는 날짜 칸에 값이 입력된 개수를 기준으로 계산

파일 분할 기능 구현 내용
- 다수의 엑셀 파일 업로드 가능
- 컬럼 행 번호 입력 가능
- 분할 방식 3가지 구현

1. 행 기준 분할
- 사용자가 입력한 행 수 단위로 분할
- 예: 1000 입력 시 1000행씩 분할
- 각 업로드 파일 기준으로 여러 개의 엑셀 파일 생성
- ZIP으로 다운로드
- 결과 파일명: split_by_rows.zip

2. 컬럼 기준 분할
- 기준 컬럼명을 입력
- 해당 컬럼 값이 같은 행끼리 하나의 파일로 묶음
- 값이 다르면 다른 파일로 분할
- 각 업로드 파일 기준으로 처리
- ZIP으로 다운로드
- 결과 파일명: split_by_column.zip

3. 시트 기준 분할
- 업로드 파일의 각 시트를 각각 별도 엑셀 파일로 분리
- ZIP으로 다운로드
- 결과 파일명: split_by_sheet.zip

서식 관련 결정 사항
- 전체 서식(색상, 폰트, 테두리, 병합 셀 등)은 유지하지 않음
- 표시형식 중심으로만 일부 유지
- 특히 날짜 셀은 원본 셀의 표시형식이 있으면 유지하도록 구현
- 숫자에 대해 천단위 콤마를 일괄 적용하는 기능은 제거함
- 현재 정책:
  - 원본 셀에 표시형식이 있으면 그것을 유지 시도
  - 일반 정수/소수는 별도 통일 서식 적용하지 않음

현재 한계
- 브라우저 기반 처리이므로 Excel의 모든 서식을 완전히 유지하지는 못함
- 병합/분할은 주로 값과 표시형식 중심
- 파일 병합 및 파일 분할의 행/컬럼 기준 처리는 기본적으로 첫 번째 시트를 기준으로 동작
- 시트 기준 분할은 전체 시트를 각각 별도 파일로 분리
